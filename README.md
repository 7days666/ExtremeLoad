# 🚀 极限负载 / ExtremeLoad

开发者工具箱，让开发准备工作一键搞定。

## 功能模块

### 📥 IDE 下载区
一键下载常用开发工具，支持多版本选择：
- Visual Studio Code
- Visual Studio 2022
- Python (3.10/3.11/3.12)
- Node.js (18/20/22)
- Git for Windows
- JDK (Eclipse Temurin 17/21)
- Go (1.22/1.23)
- Rust
- IntelliJ IDEA Community
- PyCharm Community

功能特性：
- 多版本下拉选择
- 下载进度显示（已下载/总大小）
- 取消下载、一键安装
- 自动检测已安装软件
- 搜索过滤、多任务同时下载

### 📚 运行库下载区
常用运行库一键下载安装：
- Visual C++ 运行库 (2005-2022, x86/x64)
- .NET Framework (4.0/4.5/4.8)
- msvsmon 远程调试器 (2010/2015/2019)

## 技术栈

- Python 3.10+
- PyQt5
- PyQt-Fluent-Widgets (Fluent Design UI)

## 安装

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 运行

```bash
python main.py
```
